# Human Design — Твой Дизайн

PWA приложение для расчёта бодиграфа по системе Human Design.

## Что умеет приложение

- Рассчитывает тип, стратегию, авторитет и профиль по дате и времени рождения
- Рисует бодиграф (схему центров)
- Даёт персональные рекомендации по саморазвитию
- Работает как приложение на телефоне (PWA)
- Работает офлайн после первого запуска

## Типы Human Design

- **Генератор** — ждать и реагировать
- **Манифестирующий Генератор** — реагировать, затем информировать
- **Проектор** — ждать приглашения
- **Манифестор** — информировать перед действием
- **Рефлектор** — ждать лунный цикл

## Технологии

- Чистый HTML / CSS / JavaScript (без фреймворков)
- PWA (Service Worker + manifest.json)
- Упрощённые эфемериды для расчёта позиций планет
- SVG бодиграф

## Запуск

Откройте `index.html` в браузере или разверните на GitHub Pages.

## GitHub Pages

1. Перейти в Settings → Pages
2. Source: Deploy from branch → main → / (root)
3. Сохранить

Приложение будет доступно по адресу: `https://ВАШ_ЛОГИН.github.io/human-design-app/`
